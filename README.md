### Installing
1. Configure .env file : DATABASE_URL
2. Create the database

    ```
    php bin/console doctrine:database:create
    ```
3. Make the migration

    ```
    php bin/console doctrine:migrations:migrate
    ```
4. Install the fixtures

    ```
    php bin/console doctrine:fixtures:load --append
    ```
5. Install back-end dependances

    ```
    composer install
    ```
6. Install front-end dependances

    ```
    npm install
    ```

7. Launch local server

    ```
    php bin/console server:run
    ```

8. Launch Webpack development server
   
   ```
   npm run dev-server
   ```
9. go to **http://localhost:8000**
